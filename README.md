# 📊 Layoffs Data Cleaning using SQL

## 📌 Overview

This project focuses on cleaning and preparing a real-world layoffs dataset using SQL. The dataset contained inconsistencies, duplicates, and missing values.


---

## 📂 Dataset

* Source: Kaggle
* Records: 2361 rows

---

## 🛠️ Tools Used

* MySQL
* SQL (Window Functions, Joins, Data Cleaning)

---

## 🔧 Data Cleaning Steps

* Created staging table
* Removed duplicate records using ROW_NUMBER()
* Standardized text fields (company, country, industry)
* Converted date format to SQL DATE
* Handled NULL and missing values
* Removed irrelevant rows

---

## 📊 Sample Analysis

* Total layoffs by country
* Layoffs trends over time

---

## 🎯 Key Learnings

* Practical data cleaning techniques
* Importance of data consistency
* SQL optimization and transformations
